# 辅助子载波生成测试

## 1. 测试目标

验证 `generate_aid_subcarrier` 已正确实现以下核心行为：

- 根据主载波相位与子载波倍率生成辅助子载波参考相位
- 将输入信号旋转到辅助子载波复基带
- 按 `samplerate_hz / down_samplerate_hz` 分块均值降采样
- 输出交错 `I/Q` 序列
- 对非法降采样倍率给出明确异常

## 2. 测试范围

对应文件：

- `src/signal_agent/algorithms/aid_subcarrier.py`
- `tests/algorithms/test_aid_subcarrier.py`

## 3. 测试用例

### 3.1 常量信号降采样

- 输入：全 1 复信号，主载波相位为 0，频偏为 0
- 预期：
  - 输出复基带长度等于 `down_samplerate_hz`
  - 输出复基带为全 1
  - 交错 `I/Q` 序列为 `[1, 0, 1, 0, ...]`

### 3.2 频偏与相位倍率抵消

- 输入：
  - 输入信号包含已知频偏项
  - 辅助子载波倍率为 2
  - 输入信号同时叠加 `2 * phase_carrier`
- 预期：
  - 旋转后复基带恢复为常量 1

### 3.3 非整数倍降采样约束

- 输入：`samplerate_hz / down_samplerate_hz` 不是整数
- 预期：
  - 抛出 `ValueError`
  - 异常信息包含“整数倍”
