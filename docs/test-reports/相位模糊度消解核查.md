# 相位模糊度消解核查

## 1. 核查目标

验证 `resolve_phase_ambiguity` 是否按 MATLAB `solveambiguity_qua_correct.m` 的逻辑完成相位模糊度消解，并在 `label > 0` 时执行邻近点微调。

## 2. 对齐说明

MATLAB 原逻辑分两层：

- 先以 `ratio == 1` 的支路作为参考，把其他支路的相位拉回到与参考支路相差不超过 `π` 的范围内
- 再在 `label > 0` 时，对参考支路附近的若干点做局部连续性微调，修正 `2π` 跳变

当前 Python 实现保持了这两个阶段的行为。

## 3. 测试用例

### 3.1 参考支路对齐

- 输入：`phase_correct = [0, 3.5π, -4.2π]`
- 输入：`ratio = [1, 2, 3]`
- 期望：
  - 参考索引为 `0`
  - 其他支路被修正到与参考支路相差不超过 `π`

### 3.2 邻近点微调

- 输入：带有局部跳变的相位序列
- 输入：`label = 1`
- 期望：
  - 参考支路右侧第 2 个点和更远的点按局部连续性做额外 `2π` 修正
